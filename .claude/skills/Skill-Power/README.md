# Skill-Power: Mandatory Skill Discovery & Usage

Skill-Power is a critical system skill that enforces mandatory workflows for skill discovery and usage across all Claude Code conversations.

## Purpose

- **Enforces mandatory skill discovery** before any task execution
- **Establishes non-negotiable protocols** for skill usage
- **Prevents skill skipping** regardless of perceived task simplicity
- **Ensures consistent workflow** adherence across all interactions

## Core Principle

**IF A SKILL EXISTS THAT APPLIES TO YOUR TASK, YOU HAVE NO CHOICE. YOU MUST USE IT.**

This is not optional, negotiable, or subject to interpretation.

## Key Features

- **Mandatory checklist** for all responses
- **Pattern matching** for skill discovery
- **Immediate enforcement** of skill usage
- **Clear announcement protocols** for skill usage

## Usage

This skill automatically activates at the start of conversations and establishes the foundational workflow for all subsequent interactions.

## Enforcement

- 1% chance of relevance = 100% mandatory usage
- No exceptions, no rationalizations, no workarounds
- Skills are authoritative, not suggestive

## Integration

Works with all existing skills to ensure proper discovery and usage protocols are followed consistently.
