### Query Examples

### Commands Reference
- `/log-task "description"` - Log a new task
- `/show-project-history` - Display all tasks
- `/memory-status` - Show statistics
- `/find-tasks "keyword"` - Search tasks
- `/archive-old-entries` - Clean up old records

### Best Practices
1. Always confirm task logging when prompted
2. Use descriptive task names
3. Include relevant files and notes
4. Regularly review project history
5. Archive old entries to maintain performance
